# 🔬 WSI Viewer

A frontend web application for viewing and navigating Whole Slide Images (WSI). The application provides a smooth and interactive experience for exploring high-resolution slide images through zooming and panning functionality.

## 🚀 Features

- High-Resolution Image Viewing
- Smooth Zoom In/Out
- Pan and Navigate Images
- Fast Image Rendering
- User-Friendly Controls
- Modern and Clean Design

## 🛠️ Tech Stack

- React.js
- JavaScript
- Tailwind CSS

## 🎯 About the Project

WSI Viewer is a frontend application designed to visualize large slide images efficiently. The project focuses on providing a smooth viewing experience, allowing users to explore image details through intuitive zoom and navigation controls.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/udayendra/wsi-viewer.git
```

### Navigate to the Project

```bash
cd wsi-viewer
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```
